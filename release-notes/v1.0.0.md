# GovFunding Chatbot v1.0.0 Release Notes

## 1. Release Summary
The 1.0.0 release replaces the ad-hoc prototype with a modular architecture that formalizes NSF data ingestion, documentation, and deployment readiness. It introduces a Poetry-managed ETL package, comprehensive planning docs, and automation hooks required to deliver the roadmap MVP.

## 4. Technical Details & Code Samples
```python
# etl/pipeline.py

def run_pipeline(run_date: Optional[datetime] = None, *, settings: Optional[Settings] = None) -> None:
    settings = settings or get_settings()
    run_date = run_date or datetime.utcnow()

    window_start = run_date - timedelta(days=settings.nsf_awards_window_days)

    awards_extractor = NSFAwardsExtractor(settings=settings)
    opportunities_extractor = GrantsXMLExtractor(settings=settings)

    awards_raw = awards_extractor.fetch(window_start=window_start, window_end=run_date)
    opportunities_raw = opportunities_extractor.fetch()

    awards_normalized = AwardTransformer(settings=settings).transform(awards_raw)
    opportunities_normalized = OpportunityTransformer(settings=settings).transform(opportunities_raw)

    writer = LocalWriter(settings=settings)
    writer.write_awards(awards_normalized)
    writer.write_opportunities(opportunities_normalized)
```
- Extractors live under `etl/sources/` and encapsulate network IO + raw snapshot persistence.
- Transformers in `etl/transformers/` convert external payloads into strongly typed models defined in `etl/models.py`.
- Writers handle output targets (currently local JSON; extendable to Supabase/S3).
